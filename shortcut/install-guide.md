# CtoP iOS ショートカットの作成と配布手順

このガイドは iPhone 実機で行います。完成すると「カメラ起動 → 撮影 → Apple Vision OCR → CtoP アプリに自動追記」がホーム画面アイコン 1 タップで動くようになります。

## 前提

- CtoP が `https://yucchiz.github.io/ctop/` で公開済み（Stage 8 完了後）
- iPhone に **ショートカット** アプリが入っている（標準アプリ）

## 作成手順

### 1. 新規ショートカットを作成

1. **ショートカット** アプリを開く
2. 右上の `+` → 「新規ショートカット」

### 2. アクションを追加（4 つ）

画面下の検索バーから以下を順に追加：

#### アクション 1: 写真を撮る
- 検索: 「写真を撮る」
- 追加後、アクションを**展開（▼ を押す）**
- **「カメラロールに保存」を OFF** にする ★重要★
- 「カメラ」: 背面 / 「プレビューを表示」: OFF でも可

#### アクション 2: 画像からテキストを抽出
- 検索: 「画像からテキストを抽出」または「Extract Text from Image」
- 「画像」入力に自動で「写真を撮る」の出力が入る
- 言語を**日本語**に設定（展開して確認）

#### アクション 3: クリップボードにコピー
- 検索: 「クリップボードにコピー」
- 入力に自動で「画像からのテキスト」が入る

#### アクション 4: URL を開く
- 検索: 「URL を開く」
- URL 欄に以下を入力:
  ```
  https://yucchiz.github.io/ctop/?from=shortcut
  ```

### 3. ショートカット名とアイコン

- 画面上部のタイトルをタップ → 「名前を変更」
- 名前: **「📝 CtoP スキャン」** など
- 「アイコンを選択」でカラーとグリフを設定（カメラアイコン推奨）

### 4. ホーム画面に追加

- ショートカット編集画面右上の ⓘ → 「ホーム画面に追加」
- カスタムアイコン・名前を確認して「追加」

### 5. 実行確認

- ホーム画面の「📝 CtoP スキャン」アイコンをタップ
- カメラが起動 → 紙面を撮影
- 自動で CtoP が開き、「追記しますか？」プロンプトが表示される
- 「追記」をタップで完了

## iCloud 共有リンクの取得

他の iPhone（家族や別端末）にも配布する場合：

1. ショートカットアプリで該当ショートカットを長押し
2. 「共有」→「iCloud リンクをコピー」
3. 取得した `https://www.icloud.com/shortcuts/...` を README や PWA のヘルプに貼る
4. リンクを受け取った人がそれを iPhone で開けば、ワンタップで追加可能

## プライバシー確認

このショートカットは以下の性質を持ちます：

- **写真はカメラロールに保存されません**（アクション 1 で「保存」OFF にしたため）
- **iCloud 写真にも同期されません**（保存していないため）
- **OCR 処理は端末内で完結**（Apple Vision、オフラインでも動作）
- **テキストのみがクリップボードに入り、PWA の IndexedDB に保存**される

## トラブルシューティング

| 症状 | 原因 | 対処 |
|---|---|---|
| PWA は開くが追記プロンプトが出ない | URL に `?from=shortcut` が無い | アクション 4 の URL を確認 |
| 文字認識の精度が悪い | 手ブレ・照明・角度 | 再撮影、または iOS 純正カメラのライブテキストで手動コピー |
| 毎回許可ダイアログが出る | iOS のクリップボード制約 | 仕様。「貼り付け」を許可するだけ |
| ショートカット実行時に写真がカメラロールに残る | 「保存」が ON になっている | アクション 1 を展開して「カメラロールに保存」を OFF に |
